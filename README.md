# PecubeGUI
Graphical user interface for Pecube.

Version: 1.0.0
Status: under review

## Description 
PecubeGUI is a user interface designed to work with Pecube software (version > 4.3). It enables Pecube users to design and visualize their Pecube models interactively, checking their Pecube input parameters, and plot Pecube model output.
